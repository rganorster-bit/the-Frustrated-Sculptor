Create a complete mobile-friendly web game called "הפסל המתוסכל" (The Frustrated Sculptor). The game must be entirely in Hebrew (RTL layout) and inspired by the Funmigo party game. GAME FLOW: 1. There are 2 teams: Team 1 and Team 2. 2. One player is the "Sculptor". 3. The sculptor receives a secret situation. 4. The sculptor must arrange the other team members into a human statue that represents the situation. 5. Talking is not allowed. 6. The host clicks "התחל טיימר" when the sculptor starts. 7. The timer runs until someone in the team correctly guesses the situation. 8. The host clicks "ניחשו נכון" to stop the timer. 9. Save the time for Team 1. 10. Display a new random situation for Team 2. 11. Repeat the process. 12. Save Team 2's time. 13. At the end, display both times side by side. 14. Highlight the winning team with the fastest time. 15. Show a celebration message for the winner. 16. Add a "משחק חדש" button to reset everything. REQUIREMENTS: - Use HTML, CSS and JavaScript only. - Everything must be contained in a single HTML file. - Fully responsive for phones, tablets and desktops. - Use Funmigo colors: - Yellow (#FFD400) - Black (#000000) - White (#FFFFFF) - Blue (#1E88E5) - Modern, playful and professional design. - Large buttons suitable for game hosts. - Large timer display. - RTL support. - Do not repeat situations during the same game. - Include smooth animations and visual effects.
